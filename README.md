# Note to Markdown

A Sublime Text plugin to easily save markdown-formatted txt as a tagged markdown file.

## Installation

### Sublime Package Control
* Open the command palette, either in the tools menu or by using the Command + Shift + P (Alt + Shift + P for Windows) shortcut
* Select _Package Control: Install Package_
* Look for Note2Markdown and install it

## Manually
* Create a `note_to_markdown` directory in the Sublime Packages directory
    * Locate this directory by using the _Settings... -> Browse Packages..._ menu item.
* Copy all the files and directories into this directory

## Usage
Click on _Tools -> Note to Markdown_ menu item

A popup box will have the formatted filename pre-filled. Choose the directory where you want to save the file and click 'Save'.

### Notes
The first line of the file must be the top header (`#`).

The file type must be "Plain Text".

Markdown syntax is not checked.
